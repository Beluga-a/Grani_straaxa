# 🤖 Telegram-бот «ГРАНИ СТРАХА»

Бот для редактирования квестов через Telegram. Общается с сайтом по API.

## Быстрый старт

### 1. Создай Telegram-бота
1. Открой **@BotFather** → `/newbot`
2. Скопируй **TOKEN**

### 2. Узнай свой Telegram ID
Открой **@userinfobot** → нажми Start → скопируй **ID** (число).

### 3. Установи зависимости
```bash
pip install -r requirements.txt
```

### 4. Настрой переменные
Скопируй `.env.example` → `.env` и заполни:

```
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=твой_telegram_id
SITE_URL=http://localhost:3000
ADMIN_SECRET=любая_длинная_строка
```

⚠️ **`ADMIN_SECRET` должен совпадать** с тем, что указан в `.env.local` сайта.

### 5. Запусти

Сначала сайт (в папке проекта):
```bash
npm run dev
```

Потом бот:
```bash
python bot/admin_bot.py
```

## Использование

1. `/start` — список квестов
2. Выбери квест → появятся кнопки всех полей
3. Нажми на поле → бот покажет текущее значение
4. Отправь новое значение → бот сохранит в `data/quests.json`

## Редактируемые поля

| Поле | Описание |
|------|----------|
| `name` | Название квеста |
| `desc` | Краткое описание (для карточки) |
| `full` | Полное описание (для модалки) |
| `basePrice` | Базовая цена (за группу) |
| `baseUpTo` | Сколько человек в базовой цене |
| `extraPrice` | Доплата за каждого следующего |
| `fear` | Уровень страха 1–5 |
| `diff` | Сложность 1–5 |
| `atmosphere` | Пункты атмосферы (по одному на строке) |
| `included` | Что входит в стоимость |
| `tags` | Теги/особенности |
| `schedule` | Расписание (текст) |

## На продакшн-хостинге

Поменяй `SITE_URL` на реальный домен:
```
SITE_URL=https://granistrakha.ru
```

Бота можно запустить на любом сервере (Railway, Render, VPS).
